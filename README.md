# Core-Data-Structures
Implementation of core data structures (Linked List, Stack, Queue, BST) in C++ with insert, delete, search operations and complexity analysis.
# Core Data Structures in C++

## Implemented Data Structures
- Linked List
- Stack
- Queue
- Binary Search Tree (BST)

## How to Compile

g++ linked_list.cpp -o ll  
g++ stack.cpp -o stack  
g++ queue.cpp -o queue  
g++ BST.cpp -o bst  

## How to Run

./ll  
./stack  
./queue  
./bst  

## Complexity

### Linked List
- Insert: O(1)
- Delete: O(n)
- Search: O(n)

### Stack
- Push: O(1)
- Pop: O(1)
- Search: O(n)

### Queue
- Enqueue: O(1)
- Dequeue: O(1)
- Search: O(n)

### BST
- Insert: O(log n) avg
- Delete: O(log n)
- Search: O(log n)
